The Museum

(University project, first year, second semester)

ในภาคการศึกษานี้จุดประสงค์หลักที่อาจารย์ต้องการคือ ต้องการให้เข้าใจพื้นฐานการเขียนภาษาแบบ OOP กลุ่มของพวกเราจึงเลือกทำเกมที่ไม่ซับซ้อนมาก และ scope เกมไม่ใหญ่จนเกินไป
ในเกมนี้ผู้เล่นจะต้องหลบหลีกสิ่งกีดขวางที่จะเลื่อนเข้ามาหาตัวผู้เล่น โดยผู้เล่นสามารถยิงเพื่อทำลายสิ่งกีดขวาง และเก็บไอเท็มที่เป็นประโยชน์ต่อตัวผู้เล่นได้

Engine : Visual Studio Windows Forms Application
